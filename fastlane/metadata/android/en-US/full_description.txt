Fury F-Droid Repository is an automated, ARM-only F-Droid repository.

Features:
- 34+ popular open-source Android apps
- Auto-updates every 6 hours
- ARM-only builds (arm64-v8a, armeabi-v7a)
- All APKs from official GitHub Releases
- No x86 builds - optimized for mobile devices

Apps include Mihon, Anikku, Termux, Obtainium, Neo Store, Droid-ify, CloudStream, Mastodon, MPV and many more!

Repository URL: https://UnTamed-Fury.github.io/fury-fdroid-v2/repo
